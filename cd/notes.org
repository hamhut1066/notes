given inputs x, y, a & b. define f(x, y, a, b) such that f delivers x when a = b and y when a != b

f = ( a == b) ? x : y;

f = (((a.b) + (~a.~b)).x) + (((a.~b) + (~a.b)).y)

the above two function definitions are equivalent.
